# Polices Connect Now

Déposez ici les fichiers de polices de la charte graphique :

| Fichier | Usage |
|---------|-------|
| `Staalitches-Regular.woff2` | Titres |
| `RoobertPro-Regular.woff2` | Corps de texte (400) |
| `RoobertPro-Medium.woff2` | Corps de texte (500–700) |

Une fois les fichiers ajoutés, activez les blocs `localFont` dans `lib/fonts.ts` et retirez les polices Google (Syne + Plus Jakarta Sans).
